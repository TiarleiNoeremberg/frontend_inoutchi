Política de Privacidade

Saiba como os dados pessoais são tratados no Sistema INOUTCHI, em total conformidade com a Lei Geral de Proteção de Dados (LGPD).

Quem Somos

INOUTCHI TECNOLOGIA EDUCACIONAL

Empresa responsável pelo desenvolvimento e operação do sistema INOUTCHI, plataforma tecnológica voltada à gestão educacional, controle de acesso, comunicação escolar e funcionalidades correlatas, disponibilizada na modalidade Software como Serviço (SaaS).

Papéis no Tratamento de Dados

🏫 Instituição de Ensino Controladora dos dados pessoais dos alunos, pais/responsáveis e colaboradores.
⚙️ INOUTCHI Operadora dos dados pessoais, atuando conforme as instruções da instituição de ensino e os contratos firmados.

Dados Pessoais Tratados

O sistema INOUTCHI poderá tratar, conforme configuração e uso pela instituição de ensino:

  • Dados de identificação (nome, data de nascimento)
  • Dados escolares (turma, série, registros acadêmicos operacionais)
  • Registros de entrada e saída
  • Dados de responsáveis legais (nome, CPF, telefone, e-mail)
  • Dados de acesso ao sistema (logs, data e hora de acesso)
  • Informações necessárias à comunicação escolar

🔒 Não são coletados dados para fins publicitários ou comerciais.

Finalidade do Tratamento

Os dados pessoais são tratados exclusivamente para:

  • Gestão da rotina escolar
  • Segurança e controle de acesso
  • Comunicação entre escola e responsáveis
  • Cumprimento de obrigações legais
  • Operação e melhoria contínua do sistema

Bases Legais

O tratamento de dados pessoais ocorre com fundamento nas seguintes bases legais da LGPD: execução de contrato, cumprimento de obrigação legal, consentimento do titular (quando aplicável) e legítimo interesse.

👶 Para dados de crianças e adolescentes, aplica-se o artigo 14 da LGPD, que exige consentimento específico de pelo menos um dos pais ou responsável legal.

Compartilhamento de Dados

Os dados pessoais poderão ser compartilhados com autoridades públicas, mediante obrigação legal ou ordem judicial.

🚫 Não há venda ou compartilhamento de dados para fins comerciais.

Armazenamento e Segurança

A INOUTCHI adota medidas técnicas e administrativas para proteger os dados pessoais, incluindo controles de acesso e monitoramento. Os dados podem ser armazenados em ambientes de computação em nuvem, respeitando padrões de segurança.

Direitos dos Titulares

Os titulares de dados podem exercer seus direitos (acesso, correção, eliminação, portabilidade, entre outros) junto à instituição de ensino, que é a Controladora dos dados.

Retenção e Eliminação dos Dados

Os dados serão mantidos enquanto houver vínculo com a instituição de ensino ou para cumprimento de obrigações legais, sendo eliminados ou anonimizados após esse período.

Cookies

O sistema poderá utilizar cookies estritamente necessários ao seu funcionamento e segurança, não sendo utilizados cookies para fins publicitários.

Alterações desta Política

Esta política poderá ser atualizada. A versão mais recente estará sempre disponível no sistema ou no site institucional.

Contato

Para dúvidas relacionadas a esta Política de Privacidade, o contato deverá ser realizado por meio da instituição de ensino responsável.
